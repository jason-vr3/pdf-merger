# pdf-merger
PDF 高效合并工具 (PDF Merger)：轻量级、无广告、纯本地运行的 PDF 合并工具

## ✨ 核心功能

- **批量导入**：支持一次性导入最多 128 个 PDF 文件。
- **自由排序**：在输入列表中，支持通过**鼠标按住拖拽**来自由调整 PDF 文件的合并顺序。
- **双击预览**：双击列表中的文件，可直接调用 Windows 系统默认应用打开该 PDF 进行确认。
- **智能输出**：合并后的文件会自动保存在第一个输入文件所在的目录，并自动命名为 `原文件名(合并).pdf`，不破坏原文件。
- **纯本地运行**：无需断网，所有处理均在本地完成，保障您的文档隐私与数据安全。

## 🚀 快速使用 (无需配置环境)

如果您只需要使用本软件，无需安装 Python 环境：
1. 前往右侧的 **[Releases](#)** 页面（发布页）。
2. 下载最新版本的 `pdf_merger.exe` 文件。
3. 双击直接运行即可。


Bty：本项目使用 Python + PyQt5 进行界面开发，并使用 pypdf 库来处理PDF的合并。
PyQt5 在 Windows 下有着极佳的原生体验，且支持非常方便的列表拖拽排序
